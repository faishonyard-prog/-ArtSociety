# Mahi Arts Website

Handcrafted custom portraits, mesmerizing resin pieces, and bespoke gifts created with passion and soul.

## Features
- **Custom Art Commission**: Integrated multi-step order flow for custom portraits.
- **Shop Gallery**: Browse and purchase ready-to-ship handmade creations.
- **Admin Dashboard**: Secure portal for managing products, orders, and users.
- **Modular Architecture**: Clean, scalable codebase built with React and Tailwind CSS.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop.

## Tech Stack
- **Frontend**: React (Vite)
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Deployment**: Vercel ready

## Getting Started

1. **Clone the repository**:
   ```bash
   git clone <your-repo-url>
   cd art-society
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run locally**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```

## Environment Variables
Create a `.env` file based on `.env.example` (if any API keys are added in the future).

## Deployment
This project is configured for seamless deployment on Vercel. Simply push to GitHub and connect your repository to Vercel.
